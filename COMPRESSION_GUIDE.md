# 媒體檔案壓縮教學指南 (Media Compression Guide)

本指南將教你如何使用 `ffmpeg` 提速網頁加載速度，確保你的影片同圖片喺唔損害畫質嘅情況下保持輕巧。

---

## 🛠️ 第一步：安裝工具 (FFmpeg)

如果你係 Mac 用戶，請打開 **Terminal** (終端機) 並執行以下指令安裝：

```bash
brew install ffmpeg
```

安裝完成後，可以輸入 `ffmpeg -version` 檢查是否成功。

---

## 🎞️ 影片壓縮 (Video Compression)

對於網頁背景或作品展示影片，建議將檔案控制在 **1MB - 3MB** 以內。

### 核心指令：
```bash
ffmpeg -i "原始檔案.mp4" -vcodec libx264 -crf 28 -preset faster -an "壓縮後檔案.mp4"
```

### 參數解說：
- **`-crf 28`**: 控制畫質與體積。數值越大 (例如 30) 檔案越細；數值越小 (例如 24) 畫質越好。
- **`-an`**: **(推薦)** 如果影片唔需要音效，加上呢個掣可以慳返好多位。
- **`-preset faster`**: 壓縮速度，一般用 `faster` 已經足夠。

---

## 🖼️ 圖片壓縮 (Image Compression)

雖然網頁後台已有自動壓縮功能，但如果你想喺上傳前手動優化，可以使用：

```bash
ffmpeg -i "input.jpg" -qscale:v 2 "output.jpg"
```
- `-qscale:v 2`: 畫質等級 (1-31)，2 代表高畫質，5-10 代表中等壓縮。

---

## 💡 小貼士 (Tips)

1. **命名習慣**：建議將壓縮後嘅檔案加上 `-min` 後綴 (例如 `hero-bg-min.mp4`)，方便自己辨認邊個係已優化版本。
2. **預覽效果**：壓縮完後請務必打開新檔案睇下畫質係咪可以接受。
3. **移除語氣/音軌**：網頁作品集通常係自動播放 (Muted)，所以移除音軌 (`-an`) 係最有效嘅減肥方法。
